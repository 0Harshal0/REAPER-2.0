Welcomer System with Auto-Role
All these are customizable per Server.
Total Commands: 13
Variables used: welcome, bye, joinmsg, leavemsg, autorole, joinstate and autorolestate.
Values for all the Variables must be empty except for joinstate and autorolestate.
Value for both joinstate and autorolestate: on/off
The File Consists of 4 State Commands, 6 Setup Commands and 3 Receiver Commands
-------------------------------------------------
State Commands:

1). Turn Welcomer On Command:
Trigger: (prefix)joinlogs on
Code:
$nomention
$onlyIf[$getServerVar[welcome]>0;You need to Setup the Welcome Channel and the Welcome Message First!]
$setServerVar[joinstate;on]
$title[Success!]
$description[Welcomer System has been enabled!]
$color[$random[15;99999]]

2). Turn Welcomer off Command:
Trigger: (prefix)joinlogs off
Code:
$nomention
$onlyIf[$getServerVar[welcome]>0;You need to Setup the Welcome Channel and the Welcome Message First!]
$setServerVar[joinstate;off]
$title[Success!]
$description[Welcomer System has been disabled!]
$color[$random[15;99999]]

3). Start Autorole Command:
Trigger: (prefix)autorole on
Code:
$nomention
$onlyIf[$getServerVar[autorole]>0;You need to Setup the Autoroles First!]
$setServerVar[autorolestate;on]
$title[Success!]
$description[Autorole has been enabled!]
$color[$random[15;99999]]

4). Stop Autorole Command:
Trigger: (prefix)autorole off
Code:
$nomention
$onlyIf[$getServerVar[autorole]>0;You need to Setup the Autoroles First!]
$setServerVar[autorolestate;off]
$title[Success!]
$description[Autrole has been disabled!]
$color[$random[15;99999]]

-----------------------------------------------

Setup Commands:
1). Setup Welcome Channel:
Trigger: (prefix)setjoin
Code: 
$nomention
$onlyAdmin[Let admins/mods Set the Welcome channel!]
$argsCheck[>0;says a channel!]
$title[New Log Channel]
$description[Set the Welcome channel to <#$mentionedChannels[1]>.]
$footer[Logging Commands]
$color[$random[15;99999]]
$setServerVar[welcome;$mentionedChannels[1]]
$deletecommand

2). Setup Goodbye Channel:
Trigger: (prefix)setbye
Code:
$nomention
$onlyAdmin[Let admins/mods Set the Leave channel!]
$argsCheck[>0;says a channel!]
$title[New Log Channel]
$description[Set the Leave channel to <#$mentionedChannels[1]>]
$footer[Logging Commands]
$color[$random[15;99999]]
$setServerVar[bye;$mentionedChannels[1]]
$deletecommand

3). Setup Welcome Message:
Trigger: (prefix)joinmsg
Code:
$nomention
$argsCheck[>1;Enter a Message!]
$onlyAdmin[Only Admins can access this command]
$setServerVar[joinmsg;$message]
$title[LOGS]
$description[Join Message updated by **$username**:
$message
]
$footer[Logging Commands]
$color[$random[15;99999]]

4). Setup Goodbye Message:
Trigger: (prefix)leavemsg
Code:
$nomention
$argsCheck[>1;Enter a Message!]
$onlyAdmin[Only Admins can access this command]
$setServerVar[leavemsg;$message]
$title[LOGS]
$description[Leave Message updated by **$username**:
$message
]
$footer[Logging Commands]
$color[$random[15;99999]]

5). Setup Auto-Role
Trigger: (prefix)autorole
Code: 
$nomention
$deletecommand
$argsCheck[>1;Please Mention a role]
$setServerVar[autorole;$mentionedRoles[<]]
$onlyAdmin[Only admins can access this command]
$title[ROLES]
$description[ðŸ› Auto-role has been updated to <@&$mentionedRoles[<]>]
$footer[Logging Commands]
$color[$random[15;99999]]

6). To Stop all these Logs:
Trigger: (prefix)stoplogs
Code: 
$nomention
Reseting and Stopping the Welcomer and Autorole System......
$resetServerVar[welcome]
$resetServerVar[bye]
$resetServerVar[joinmsg]
$resetServerVar[leavemsg]
$resetServerVar[autorole]
$resetServerVar[joinstate]
$resetServerVar[autorolestate]
$editIn[2s;The Welcomer and the AutoRole System has been Sucessfully Stopped!]

Receiver Commands:
1). Receiving Welcome Message
Trigger: $onJoined[$getServerVar[welcome]]
Code:
$nomention
$onlyIf[$getServerVar[welcome]>0;]
$title[WELCOME!]
$description[<@$authorID>, $getServerVar[joinmsg]]
$footer[Logging]
$color[#ff2052]

2). Receiving Goodbye Message
Trigger: $onLeave[$getServerVar[bye]]
Code:
$nomention
$title[GOODBYE!]
$onlyIf[$getServerVar[leave]>0;]
$description[<@$authorID> $getServerVar[leavemsg]]
$footer[Logging]
$color[#ff2052]

3). Giving Auto-Role to New Joiners
Trigger: $onJoined[$getServerVar[autorole]]
Code:
$nomention
$onlyIf[$getServerVar[autorole]>0;]
$giveRole[$getServerVar[autorole];$authorID]
